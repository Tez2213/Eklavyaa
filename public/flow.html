Technology Stack & Data Flow

Kodular WebView container for native Android app experience
Next.js 13+ with App Router for unified platform and games
Progressive Web App with Service Workers for offline capability
IndexedDB for persistent local data storage and progress tracking
Cache API for storing game assets, images, and educational content
Background Sync for seamless online/offline data synchronization
Vercel hosting with global CDN for fast content delivery
React 18 with Framer Motion for smooth animations and interactions
